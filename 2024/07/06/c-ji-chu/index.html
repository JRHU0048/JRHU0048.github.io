常量

  • 字面常量
    • 把常量书写到代码内部，叫字面常量
    • 可以有整数（整型）、小数（实型），二者统称为数值型常量
    • 字符型字面常量：由单引号包围的单个字符
    • 字符串字面常量：以双引号包围的，任意个数个字符
  • 符号常量
    用标识符定义的常量，给常量一个名字，就是符号常量
    • 在c语言中使用编译预处理指令#define定义符号常量：#define PI 3.14
    • C++中在类型前添加关键词const来定义：const double PI=3.14 const int N = 6
    • 符号常量的值必须在定义时指定，且不能赋新值
